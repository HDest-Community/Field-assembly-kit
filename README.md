### Notes
---
- Weapon upgrade kits can only be found in backpacks. They are also sold by the [Merchant](https://gitlab.com/accensi/hd-addons/merchant) if you have that loaded.
- Loadout code is `wuk`.

### Supported Mods
---
- Peppergrinder
- ZM69
- Bossmerg

NOTE: Vanilla weapons are supported by default.
